Web Developer

Computer design is the most relevant and growing artform in the world.

 My goal is to develop aesthetically great, functioning online sites. I focus on the aesthetic design of the page as well as the users-interaction with the page. I try to maintain a high level of creativity while making the page managable to new users. I take this approach with all my projects running multiple tests and constantly updating work. It is also of the upmost importance to make the sites cross compatible across deferent platforms and browsers. For some projects there may be multiple designs just to accommodate to certain devices.

This industry is always changing and I wish to adapt with it as it grows; in order to produce the most technologically advanced and naturally flowing web experiences possible. I want to put my skills to good use designing all sorts of sites from fun creative sites to more straightforward corporate style sites. It’s my mission in life to create memorable online experiences and look forward to working with anyone who can help me achieve this goal.

TOOLS

Languages and frameworks I've used recently:

Of course I always use Html and CSS, the main languages of the web.

I Also utilize the simplicity of Sass.

JavaScript is my favorite tool for the web.

It's works so naturally with web pages and it seems like the possibilities for this language are endless. Along with Jquery, I use Javascript on all my web projects.

MeteorJs is my favorite javascript framework and package manager.

After only a three months of working with meteor it has quickly become my favorite javascript frame work because of the wide range of possibilities it offers. It works as a framework, a package manager, and an appbuilder.

Angular is another framework I love to utilize.

Angular works great with meteor, the best part about angular is it the additional plug-ins it has. Some of the angular plug ins I've used are ng-cordova for mobile app development, angular-ui-router, and angular-animate.

I'm a Creative Suite Junkie.

I've used Adobe programs to create and design content for the past 7 years. The primary tools from creative suite that I use for every project are Illustrator and Photoshop. I also use Edge Animate to create some great animations that are written in Javascript. I only use Dreamweaver for it's FTP tool but Adobe Brackets is my primary text editor.

Though Meteor has replaced Yeoman as my primary build tool I still use Yeoman to develop great compiled HTML E-mails.

I love using bootstrap as a CSS framework. It makes it really easy for me to create great responsive designs. I use bootstrap on almost all of my projects as well.
